# RHEL 10 Compliance as Code

Automated security hardening and compliance assessment for Red Hat Enterprise Linux 10 using OpenSCAP and Ansible.

## Overview

This project demonstrates an automated Compliance as Code approach to audit and harden a RHEL 10 workstation against the CIS Benchmark Level 1 profile using OpenSCAP for security scanning and Ansible for automated remediation.

**Results:** Compliance score improved from **77.8%** to **98.2%** through automated remediation.
